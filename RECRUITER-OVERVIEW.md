# Recruiter Overview — Portfolio técnico

## Perfil técnico

Portfolio técnico orientado a sistemas, SQL Server, infraestructura Windows, alta disponibilidad, operación IT y seguridad defensiva.

El repositorio está construido como una colección de laboratorios documentados, con arquitectura, pruebas, incidencias, evidencias y conclusiones técnicas. El objetivo es mostrar trabajo práctico defendible en entrevista.

## Competencias principales

- Administración de sistemas Windows Server y Windows 11.
- Active Directory, DNS, OUs, grupos y cuentas de servicio.
- SQL Server DBA: instalación, configuración, backup, recovery, mantenimiento y monitorización.
- SQL Server Agent, Database Mail, alertas operativas y jobs de mantenimiento.
- SQL Server Audit, Query Store y dashboard DBA.
- Alta disponibilidad con Windows Server Failover Cluster y SQL Server Always On Availability Groups.
- Segmentación de red, pfSense, Wazuh, PAW y seguridad defensiva.
- Documentación técnica en Markdown orientada a portfolio profesional.
- Troubleshooting y validación con PowerShell, T-SQL y evidencias reproducibles.

## Enfoque del portfolio

Este portfolio está preparado para mostrar una base técnica práctica y documentada. Cada laboratorio incluye diseño, despliegue, validación, evidencias e incidencias relevantes, con el objetivo de demostrar criterio técnico y capacidad de trabajar sobre entornos realistas de sistemas, bases de datos y seguridad defensiva.

La documentación prioriza claridad, trazabilidad y resultados comprobables frente a teoría genérica.
