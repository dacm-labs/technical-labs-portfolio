# Valor profesional — LAB-01

## Resumen

El LAB-01 resume un entorno práctico de administración SQL Server orientado a portfolio técnico.

Incluye instalación, seguridad, backup, recuperación, automatización, observabilidad y documentación.

## Competencias demostradas

- Administración de Windows Server y dominio.
- Instalación y configuración de SQL Server.
- Separación de datos, logs, TempDB y backups.
- Backup, restore y recuperación.
- Jobs y schedules con SQL Server Agent.
- Seguridad basada en grupos y roles.
- Alertas operativas y revisión de eventos.
- Documentación técnica para GitHub.

## Uso en entrevistas

Este laboratorio sirve para explicar decisiones técnicas reales y demostrar trabajo práctico con evidencias.
