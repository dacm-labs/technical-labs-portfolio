# Valor profesional LAB-02

LAB-02 demuestra un escenario práctico de alta disponibilidad para SQL Server.

## Competencias demostradas

- Windows Server Failover Cluster.
- SQL Server Always On Availability Groups.
- Listener de conexión.
- Failover y failback.
- Lectura en réplica secundaria.
- Validación de DNS y puertos.
- Jobs adaptados a alta disponibilidad.
- Troubleshooting técnico.
- Evidencias visuales para portfolio.

## Uso en entrevistas

Este laboratorio permite explicar cómo se diseña, valida y documenta una arquitectura de continuidad de servicio basada en SQL Server.

## Conclusión

LAB-02 refuerza el portfolio al demostrar alta disponibilidad, operación controlada y documentación técnica defendible.
