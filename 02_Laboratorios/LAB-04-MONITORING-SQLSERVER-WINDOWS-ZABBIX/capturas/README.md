# Capturas LAB-04

## Estado

La carpeta `capturas/` se mantiene como ubicación reservada para capturas de trabajo o material local no publicado directamente en la galería final.

Las capturas seleccionadas para publicación están organizadas en:

```text
evidencias/images/
```

## Criterio aplicado

Para la versión pública del LAB-04 se decidió publicar únicamente 15 evidencias seleccionadas, evitando duplicados y capturas sin valor técnico claro.
